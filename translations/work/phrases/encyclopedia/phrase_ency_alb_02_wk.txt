ENCY_THM_02_05()
{
	[Daggers]
}
ENCY_THM_TXT_02_05()
{
	[Ces armes légères et élégantes sont les préférées des citadins. Elles permettent de porter des coups rapides, et surtout peuvent facilement être maniées de la main gauche, rendant ainsi possible le combat avec une arme dans chaque main. Cette technique est très appréciée des aristocrates et des duellistes. Les Trykers aiment particulièrement les dagues ; le gouverneur Still Wyler lui-même serait d'ailleurs expert dans leur maniement.\n
			\n
			Type : Arme de mêlée à une main.\n
			Parties de l'objet utilisées avec les plans de création : lame, hampe, poignée, contrepoids\n
			Dagues Fyros : Jab (basique), Jabus (moyen), Jabyx (supérieur)\n
			Dagues Matis : Limmel (basique), Modi Limmel (moyen), Kara Limmel (supérieur)\n
			Dagues Trykers : Fyler (basique), Lor-Fyler (moyen), Ry-Fyler (supérieur)\n
			Dagues Zoraïs : Lam (basique), Li'Lam (moyen), Zo'Lam (supérieur)\n
			Dagues spéciales de peuple : Dague vivante limmel (Matis), dague ondulante fyler (Tryker)\n
			Tribus utilisant une dague tribale : les Corsaires, les Illuminés et les Cuzans\n
			Type de dégâts : perforant (attaque spéciale : perforation des armures)\n
			Vitesse de l'arme : très rapide\n
			Dégâts de l'arme : faibles\n
			Allonge : courte\n
			Volume : faible\n
			Défense : Pas de pénalité sur les esquives, forte pénalité sur les parades\n
			Style de combat : les dagues peuvent être utilisées de la main gauche, rendant possible les combats à deux armes.\n
			Particularités : Cette arme entraîne une faible pénalité sur les actions, permettant ainsi plus de coups spéciaux pour un coût donné. Cette arme nécessite peu de matériaux pour sa création.]
}

ENCY_THM_02_08()

	[Maces]
}
ENCY_THM_TXT_02_08()
{
	[Les masses, également appelées massues, sont des armes utilisées par les homins depuis la nuit des temps. Elles sont solides et efficaces, et font partie de l'équipement de nombreux guerriers. Certains voient en elles des armes moins meurtrières, car moins promptes à faire couler le sang. C'est une vision quelque peu idéalisée, et le combattant qui voit son bouclier voler en éclat sous les coups d'une masse revient vite à la réalité. Mais peut-être est-ce cette vision qui a poussé de nombreux Zoraïs des temps anciens, se décrivant comme pacifiques, à apprendre le maniement de ces armes dans le but de se défendre contre les aggressions des barbares. C'est ainsi que la masse est devenue emblématique du peuple zoraï.\n
			\n
			Type : Arme de mêlée à une main.\n
			Parties de l'objet utilisées avec les plans de création : marteau, hampe, poignée, contrepoids\n
			Masses Fyros : Masson (basique), Massonus (moyen), Massonyx (supérieur)\n
			Masses Matis : Bristo (basique), Modi Bristo (moyen), Kara Bristo (supérieur)\n
			Masses Trykers : Tumbler (basique), Lor-Tumbler (moyen), Ry-Tumbler (supérieur)\n
			Masses Zoraïs : Nanka (basique), Li'Nanka (moyen), Zo'Nanka (supérieur)\n
			Masse spéciale de peuple : Massue électrique nanka\n
			Tribus utilisant une masse tribale : la Sève Sacrée, le Cercle Noir, les Gibads\n
			Type de dégâts : contondant (attaque spéciale : vertige, ralentit les attaques de l'adversaire)\n
			Vitesse de l'arme : lente\n
			Dégâts de l'arme : assez élevés\n
			Allonge : moyenne\n
			Volume : moyen\n
			Défense : Faible pénalité sur les esquives, pénalité moyenne sur les parades\n
			Style de combat : Peut être utilisée avec un bouclier ou une dague dans la main gauche\n
			Particularités : Pénalité infligée sur les parades de l'adversaire.]
}

ENCY_THM_TXT_02_12()
{
	[Les haches à deux mains figurent parmi les armes les plus impressionnantes des homins ; bien qu'elles soient lentes et pesantes, elles permettent de porter des coups d'une puissance dévastatrice. Les guerriers qui les utilisent ont toujours été craints sur les champs de bataille. Dans les temps anciens, les bourreaux impériaux utilisaient des haches à deux mains pour accomplir leur sinistre besogne, c'est pourquoi les maîtres combattants qui manient ces armes sont appelés exécuteurs. Parmi les homins, les Fyros sont ceux qui apprécient le plus les grandes haches, et la hache ardente retch est l'un des fleurons de leur artisanat.\n
			\n
			Type : arme de mêlée à deux mains.\n
			Parties de l'objet utilisées avec les plans de création : lame, hampe, poignée, contrepoids\n
			Haches Fyros : Retch (basique), Retchus (moyen), Retchyx (supérieur)\n
			Haches Matis : Klout (basique), Modi Klout (moyen), Kara Klout (supérieur)\n
			Haches Trykers : Weld (basique), Lor-Weld (moyen), Ry-Weld (supérieur)\n
			Haches Zoraïs : Kra-cho (basique), Li'Kra-cho (moyen), Zo'Kra-cho (supérieur)\n
			Hache spéciale de peuple : Hache ardente retch\n
			Tribus utilisant une hache tribale : les Percécorces, les Pillards du Désert, les Veilleurs\n
			Type de dégâts : tranchant (attaque spéciale : hémorragie)\n
			Vitesse de l'arme : très lente\n
			Dégâts de l'arme : très élevés\n
			Allonge : longue\n
			Volume : important\n
			Défense : forte pénalité sur les esquives\n
			Style de combat : les armes à deux mains ne peuvent pas être utilisées avec un bouclier ou une dague dans la main gauche\n
			Particularités : cette arme difficile à manier entraîne une pénalité conséquente sur les actions de celui qui l'utilise. De par la puissance de ses coups, elle inflige également une pénalité à l'adversaire, qui voit ses chances de parade diminuées. En revanche, les attaques à la hache sont lentes, et donc plus faciles à esquiver.]
}
